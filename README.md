# HelloBooks Restaurant MVP

A customized front-end for restaurant accounting software, built as part of the HelloBooks AI platform.

## Features

- **Dashboard** - Real-time revenue tracking, food cost %, and order analytics
- **Menu Costing** - Recipe cost calculation with ingredient breakdown
- **Inventory Management** - Stock tracking with low-stock alerts
- **AI Assistant** - Smart insights and natural language queries

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
